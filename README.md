# Whydah-openidconnect-testfrontend
Simple node app used to verify login with openid-connect


## Getting started

*1*
```
cp .env_template .env
npm install
npm start
```
*2*
http://localhost:3000/

*3*
Login with Entra gpxx@entra.no user
